Integración PayPal



  
  

  
Preguntas:

  
    
    
  • ¿Qué API seleccionaste? PayPal REST API.
    • 
    
  • ¿Qué información obtienes del API? Se obtiene un ID de orden de pago que puede ser usado para capturar dinero.
    • 
    
  • ¿Cómo integraste la respuesta en tu aplicación web? Usando Express para hacer llamadas HTTP a PayPal, y luego mostrar los datos con HTML y JavaScript.
    • 
    
  • ¿Qué retos enfrentaste al consumir el API? Configuración de credenciales y entender cómo funciona el flujo de autorización.
    • 
    
  • ¿Qué aprendiste al hacerlo? Cómo autenticarme con OAuth2 y crear pagos con la API de PayPal usando el entorno sandbox.
    • 
  
  
  


Ventajas y Desventajas de integrar servicios web de terceros


Ventajas:


    
  
  • Permite ahorrar tiempo de desarrollo al reutilizar funcionalidades ya hechas.
    • 
  
  • Acceso a tecnologías avanzadas (como pagos, mapas, autenticación) sin tener que implementarlas desde cero.
    • 
  
  • Actualizaciones y mantenimiento corren por cuenta del proveedor del API.
    • 
  
  • Escalabilidad: muchos servicios están preparados para alto tráfico.
    • 




Desventajas:


    
  
  • Dependencia de un tercero: si el API deja de funcionar, tu app puede verse afectada.
    • 
  
  • Limitaciones de uso (cuotas, restricciones de sandbox, etc.).
    • 
  
  • Requiere comprender bien la documentación del API.
    • 
  
  • Posibles costos adicionales si se excede el uso gratuito.
